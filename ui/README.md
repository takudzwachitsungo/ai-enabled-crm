# AI-Enabled CRM UI

This frontend starter is now shaped for the backend platform in this repository rather than the original template demo.

## Current Scope

- Command-center overview aligned to backend readiness
- CRM core, service, marketing, commerce, AI, and audit workspaces
- Theme and information architecture designed around the project’s actual modules

## Getting Started

1. Run `npm install`
2. Run `npm run dev`
3. Start wiring real API calls into the current workspace views
